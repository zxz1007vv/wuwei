1st Shinei Select

The Shinei Select tournament (新鋭選抜, "fresh top-selection") was held once, in 1962. It was sponsored by the Asahi Shinbun. Time was 5h each, komi 4.5. See also Kido Yb. 1963, p. 165. Some sources call this the "Asahi 5,6-dan tournament".

Four New Stars League

In 1962 Kido organized the Four New Stars League (新鋭四棋士リーグ戦). Rin Kaiho won all his games.
